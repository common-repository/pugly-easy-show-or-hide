== Description ==

Easily show or hide any content from logged in or logged out users simply by assigning a class!

== Changelog ==

= 1.0.0 =
* Added the ability to hide content for logged in users.

== Installation ==

1. Install easily with the WordPress plugin control panel or manually download the plugin and upload the folder to the /wp-content/plugins/ directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. To hide content for logged in users assign the class 'pugly-hide-for-logged-in' to the item you wish to hide
4. To hide content for logged out users assign the class 'pugly-hide-for-logged-out' to the item you wish to hide
